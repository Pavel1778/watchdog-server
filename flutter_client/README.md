# 🐕 Watchdog Server v2.0

Node.js сервер системы видеонаблюдения с поддержкой нескольких доноров, JWT-аутентификацией, двусторонней аудиосвязью и автозаписью.

---

## ✨ Что нового в v2.0

| Функция | Описание |
|---|---|
| 🔐 JWT Auth | Аутентификация с ролями admin/viewer |
| 👁️ Мультидонор | Несколько телефонов-доноров одновременно |
| 📹 Авторекординг | Запись начинается автоматически с первого кадра |
| 🎤 Двусторонний звук | Наблюдатель слышит донора и может говорить в ответ |
| 🌐 Веб-интерфейс | Полноценный интерфейс наблюдателя в браузере |
| ⏹️ Контроль стрима | Наблюдатель может остановить/запустить стрим донора |
| 🔦 Фонарик | Управление фонариком каждого донора |
| 👮 Adminпанель | Статистика сервера, удаление клипов (пароль: 1379) |

## 📡 API Endpoints

### Публичные
| Метод | URL | Описание |
|---|---|---|
| GET | `/status` | Статус сервера и список доноров |
| POST | `/stream/frame?donorId=X` | Донор: отправить JPEG кадр |
| POST | `/clips/upload?donorId=X` | Донор: загрузить mp4 клип |
| WS | `/donor?donorId=X` | Донор: WebSocket канал команд |
| WS | `/observer/audio?donorId=X` | Двусторонний аудио канал |

### Требуют токена (Bearer)
| Метод | URL | Описание |
|---|---|---|
| POST | `/auth/login` | Получить JWT токен |
| GET | `/stream?donorId=X` | MJPEG видеопоток |
| GET | `/donors` | Список всех доноров |
| POST | `/torch/on?donorId=X` | Включить фонарик |
| POST | `/torch/off?donorId=X` | Выключить фонарик |
| POST | `/stream/stop?donorId=X` | Остановить стрим донора |
| POST | `/stream/start?donorId=X` | Запустить стрим донора |
| GET | `/clips` | Список записей |
| GET | `/clips/:filename` | Скачать запись |
| DELETE | `/clips/:filename` | Удалить запись (только admin) |

## 📱 Flutter Клиент (Донор)

### Зависимости (`pubspec.yaml`)
```yaml
dependencies:
  camera: ^0.10.5
  wakelock_plus: ^1.1.4           # CPU не засыпает
  flutter_foreground_task: ^7.0.0  # Фоновый сервис Android
  permission_handler: ^11.2.0      # Разрешения (батарея, камера, микрофон)
  web_socket_channel: ^2.4.0       # WebSocket
  http: ^1.1.0                     # HTTP
  record: ^5.1.1                   # Захват микрофона
  audioplayers: ^5.2.1             # Воспроизведение звука от наблюдателя
```

### Ключевые возможности клиента
- 🔋 **WakeLock** — экран может гаснуть, CPU продолжает работу
- 🔔 **Foreground Service** — постоянное уведомление, Android не убьёт приложение
- 🔋 **Отключение оптимизации батареи** — запрашивается автоматически при старте
- 📷 **Непрерывный стрим** — ~15 кадров/с даже при выключенном экране
- 🎤 **Аудио к серверу** — PCM16 через WebSocket
- 🔊 **Аудио от наблюдателя** — воспроизведение голоса наблюдателя

### Команды от наблюдателя (через WebSocket)
```json
{ "cmd": "torch",       "state": "on"  }  // включить фонарик
{ "cmd": "torch",       "state": "off" }  // выключить фонарик
{ "cmd": "stopStream"                  }  // остановить стрим
{ "cmd": "startStream"                 }  // запустить стрим
```

## 🌐 Веб-интерфейс наблюдателя

Открой в браузере: `https://your-server.onrender.com`

### Возможности
- 🔐 Логин с выбором роли (viewer / admin)
- 📺 MJPEG стрим любого донора
- 🎙️ Двусторонняя аудиосвязь (кнопка МИК)
- 🔦 Управление фонариком
- ⏹️ Стоп/старт стрима (admin)
- 📁 Список и скачивание клипов
- 📊 Статус всех доноров в реальном времени
- 🔴 Индикатор REC и счётчик FPS

---

## 🔊 Двусторонняя аудиосвязь

```
Телефон-донор ──── WebSocket /observer/audio ───▶ Сервер ───▶ Браузер наблюдателя
              ◀────────────────────────────────────────────── (голос наблюдателя)
```

**Формат:** PCM 16-bit, 16kHz, mono, кодировка base64  
**Протокол:** `{ "type": "audio", "data": "<base64>" }`

---

## 📹 Автозапись

- Запись **начинается автоматически** с первого кадра от донора
- Формат: MJPEG (можно конвертировать через ffmpeg)
- Запись **останавливается** при отключении донора или команде `/stream/stop`
- Файлы хранятся в папке `clips/`

## 🔒 Безопасность

- JWT токены подписаны случайным секретом при каждом запуске сервера
- Роли: `admin` (полный доступ) и `viewer` (только просмотр и скачивание)
- Удаление клипов — только для admin
- Остановка стрима — только для admin

---

## ⚠️ Ограничения бесплатного Render

- Клипы удаляются при рестарте (файловая система эфемерная)
- Сервис засыпает после 15 мин неактивности
- Лимит: 750 часов/месяц
